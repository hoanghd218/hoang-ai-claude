# Adult Coloring Book Prompt Guide (Cozy & Cute Style)

## Source
Derived from Hoja 1 super prompts — optimized for Nano Banana Pro on Google AI Studio / Gemini.

---

## Core Aesthetic: "Cute Cozy Medium-Detail"

Every page must feel like a complete, inviting scene — never empty, never overwhelming.

---

## Page Prompt Requirements

Each coloring page prompt MUST specify:

1. **Black-and-white line art** — finished coloring page, not a sketch. **White background, NO borders, NO frames, NO rectangular boundary lines**
2. **Medium detail** — enough to be engaging for adults, but all details must be clean, bold, and easy to color
3. **Complete layered scenes** with foreground, midground, and background elements
4. **Large, clear decorative shapes** — no dense micro-patterns
5. **Simplified vegetation** — large stylized shapes, wide line spacing, zero micro-veins, zero fine-line textures
6. **Spaced-out background motifs** — wallpapers, textiles, patterns must use big, widely spaced shapes
7. **Kawaii proportions** on characters — consistent across all pages
8. **Cozy fully developed environments** with props like:
   - Shelves, lamps, cushions, windows, curtains
   - Tables, framed art, rugs, cozy furniture
   - Gentle outdoor scenery (gardens, patios, porches)

---

## MUST AVOID in Prompts

- **Borders, frames, or rectangular boundary lines** around the image
- Dense clusters of small shapes
- Micro-veins on leaves/plants
- Fine-line textures
- Hyper-detailed botanical patterns
- Visual noise in wallpapers/textiles
- Empty or sparse compositions
- Cluttered/unreadable layouts

---

## Character Guidelines

- Kawaii proportions (large head, small body, big expressive eyes)
- Consistent style across all pages
- Main character may appear alone OR with 1-2 secondary characters
- NOT every page needs multiple characters
- All interactions must feel warm, cute, cohesive

---

## Cover Prompt Requirements

The cover prompt must specify:
- **Full-color illustration** (NOT black-and-white)
- Include the title and subtitle text
- Warm, premium cozy aesthetic
- Multiple large, readable props and decorative elements
- Must state "Coloring Book for Adults"

---

## Example Prompt Structure (for a single page)

```
Black and white line art illustration for an adult coloring book, cute cozy kawaii
aesthetic, medium detail, bold clean outlines, large open shapes for easy coloring,
no shading. NO borders, NO frames, NO rectangular boundary lines around the image.
White background.

Scene: [main character] is [doing activity] in [cozy setting].

Foreground: [table items, food, crafts — name 3-4 specific objects].

Midground: [character actions, furniture, shelves, lamps — name 3-4 specific objects].

Background: [windows with outdoor scenery, wall art, curtains — name 3-4 specific objects].

Clean bold outlines, cozy relaxing environment, easy-to-color shapes, adult coloring
book page. NO borders or frames.
```
